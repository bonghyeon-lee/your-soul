# Your Soul: The Honest Mirror (AI Mirror)

> *"거울은 정직하지만, 해석은 다정해야 한다."*

**Your Soul**은 당신과 AI의 상호작용(대화, 코드, 글)을 분석하여, 당신의 숨겨진 성향, 강점, 그리고 성장 가능성을 발견해주는 "AI 거울" 프로젝트입니다. 단순한 기록을 넘어, **'나'라는 사람을 입체적으로 이해하고 더 나은 모습으로 성장하도록 돕는 파트너**를 지향합니다.

## 🚀 프로젝트 소개 (Introduction)

우리는 매일 AI와 수많은 대화를 나눕니다. 이 대화 속에는 나의 말투, 논리 구조, 감정 상태, 그리고 해결하고자 하는 문제들이 고스란히 담겨 있습니다.
**Your Soul**은 이 데이터를 분석하여:

1. **발견 (Discovery)**: 내가 미처 몰랐던 나의 언어 습관과 사고 패턴(Blind Spot)을 찾아줍니다.
2. **기록 (Documentation)**: 나의 영혼(Soul)을 `your-soul.md`라는 살아있는 문서로 시각화합니다.
3. **성장 (Growth)**: 내가 원하는 이상적인 모습(Target Persona)으로 나아갈 수 있도록 구체적인 피드백을 제공합니다.

## ✨ 핵심 기능 (Key Features)

### 1. 실시간 성향 분석 (Real-time Analysis)

- **Communication Style**: 어조, 예의, 감정 표현 빈도 분석.
- **Work Style**: 논리적 구조, 창의성, 실행력 평가.
- **Blind Spot Detector**: 자주 사용하는 부정적 단어나 논리적 비약을 감지.

### 2. 살아있는 문서화 (Living Documentation)

- 분석된 결과는 `your-soul.md` 파일에 지속적으로 업데이트됩니다.
- Git을 통해 나의 변화 과정을 히스토리로 관리할 수 있습니다.

### 3. 목표 성장 관리 (Growth Management)

- **Desired Persona**: 내가 되고 싶은 모습을 설정합니다 (예: "따뜻한 리더", "냉철한 분석가").
- **Soul Diff**: 현재 모습과 목표 모습 간의 차이를 분석하고, 구체적인 행동 가이드를 제안합니다.

### 4. 프라이버시 우선 (Privacy First)

- 가장 내밀한 대화를 다루는 만큼, 모든 데이터는 **로컬(Local)** 환경에서 안전하게 처리됩니다.
